## 🔍 Overview
This dashboard is a **business-facing Power BI report** designed to present hotel performance in a clear and decision-friendly way.  

The goal of the dashboard is to help managers and business leaders quickly understand how hotel properties are performing across **revenue, occupancy, bookings, room categories, and booking platforms**.  

It provides a leadership view of:
- Revenue performance  
- Occupancy trends  
- Property comparison  
- Room category performance  
- Booking platform contribution  
- Property and room-level detail  

---

## 🗂️ Dataset
This dashboard was built using a **public hospitality dataset**, adapted to simulate a real-world hotel operations reporting environment.  

### 📌 Business Scenario
The dataset represents a hotel group managing multiple properties across different cities, room categories, and booking channels.  

It captures key operational and financial aspects such as:
- Hotel bookings  
- Realized revenue  
- Room capacity  
- Successful bookings  
- Cancellations  
- Room category usage  

This enables realistic analysis of **financial performance, capacity utilization, and booking behavior**.

### 📊 Key Data Fields
- Hotel properties and locations  
- Room categories and room classes  
- Booking dates and check-in dates  
- Booking platforms  
- Booking status  
- Successful bookings  
- Room capacity  
- Revenue generated and revenue realized  

---

## 🎯 Dashboard Objectives
This dashboard is designed to answer critical business questions such as:
- How is hotel revenue trending over time?  
- Which cities show stronger occupancy performance?  
- Which properties generate the most revenue?  
- Which booking platforms drive the most bookings?  
- Which room classes contribute most to revenue?  
- How do properties compare in capacity and utilization?  

---

## 📈 Key Performance Indicators (KPIs)

- **Total Revenue**  
  Total realized revenue across the selected period  

- **Total Bookings**  
  Total number of bookings in the selected view  

- **Occupancy %**  
  Percentage of available room capacity successfully booked  

- **Avg Revenue per Booking**  
  Average realized revenue per successful booking  

- **Revenue per Available Room (RevPAR)**  
  Revenue generated relative to available room capacity  

- **Cancellation %**  
  Percentage of bookings that were cancelled
